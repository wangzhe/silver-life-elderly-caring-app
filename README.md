# Silver Life Elderly Caring App

智慧养老院管理平台是一个专门为养老机构设计的机器人操作和监控平台。该平台支持多模态输入，提供实时监控、机器人管理和紧急事件处理等功能。

## 主要功能

### 1. 养老机构配置
- 基本信息配置
- 布局结构管理
- 房间信息管理
- 老人信息管理（支持多模态输入）

### 2. 养老院运营
- 实时房间监控
  - 老人状态监测（睡眠、活动、如厕等）
  - 生命体征监测（呼吸、体温、心率）
- 机器人巡查任务分配
- 预警系统（可配置阈值）

### 3. 机器人管理
- 实时状态监控
- 位置追踪
- 视频流查看
- 远程控制

## 技术栈

- 前端框架：React + TypeScript
- UI组件：shadcn/ui + Tailwind CSS
- 状态管理：React Hooks
- 图标库：Lucide Icons

## 项目结构

```
frontend/
├── src/
│   ├── components/
│   │   ├── layout/      # 布局组件
│   │   ├── monitoring/  # 监控相关组件
│   │   ├── robot/       # 机器人管理组件
│   │   └── alerts/      # 预警组件
│   ├── hooks/           # 自定义Hooks
│   ├── types/           # TypeScript类型定义
│   └── App.tsx          # 主应用组件
├── .env                 # 开发环境配置
└── .env.production      # 生产环境配置
```

## 部署流程

1. 安装依赖
```bash
cd frontend
npm install
```

2. 开发环境运行
```bash
npm run dev
```

3. 生产环境构建
```bash
npm run build
```

4. 部署
将`frontend/dist`目录下的文件部署到Web服务器。

## 界面预览

### PC端
- 左侧固定导航栏
- 顶部状态栏（用户信息、警告数、系统时间）
- 多标签页布局
- 响应式设计（>1024px自动启用）

### 移动端
- 底部导航栏
- 单页面布局
- 触屏优化交互

## 注意事项

1. 环境变量配置
- 开发环境：`.env`
- 生产环境：`.env.production`

2. 机器人控制
- 需要配置正确的机器人控制服务URL
- 视频流服务URL配置

3. 预警系统
- 支持每位老人独立配置预警阈值
- 分级预警处理流程
- 自动通知升级机制
